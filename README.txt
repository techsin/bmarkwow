The bookmark extension for chrome web browser that is designed to handle bookmarks and make them more managebale.

Features:-
1. Link Bookmarks with each others. Upto 4.
2. Auto open all previous bookmarks scan main content for most used words and tag them with those for search feature. And do periodic tagging from after the point of first installation.
3. Perodically check for broken links.
4. Ability to make custom lists out of existent bookmarks and share them.
5. Ability to sort by date, used, sitenameAlphabetically, topSites, and so on for even inside folders. (Chrome lacks ability to sort beside alphabetical order. There should be recent option in all subfolders!).

Main Features are
-ability to mark bookmarks as similar.
-make lists.
-auto tag.
